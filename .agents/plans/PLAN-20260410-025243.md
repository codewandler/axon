### Task B3: Implement `indexImplementations()` in the Go indexer

**Files modified**: `indexer/golang/indexer.go`  
**Estimated time**: 30 minutes

**What to write**:  
Add a new function `indexImplementations(ctx, ictx, moduleURI, allPkgs []*packages.Package) error`.

Algorithm:
1. Collect all interface types across all packages in the module:
   - For each pkg, iterate `pkg.Types.Scope().Names()`
   - For each name, get `obj := pkg.Types.Scope().Lookup(name)`
   - If `obj` is a `*gotypes.TypeName` and its underlying type is `*gotypes.Interface`, add to `interfaces` map keyed by URI
2. For each package, iterate its struct types:
   - For each struct, call `gotypes.Implements(structType, iface)` and
     `gotypes.Implements(gotypes.NewPointer(structType), iface)` for each interface
   - If either is true, emit `edge := graph.NewEdge(types.EdgeImplements, structNodeID, ifaceNodeID)`

Call `indexImplementations()` from `indexModule()` after the package-indexing loop, passing
`pkgs` (all loaded packages).

**Verification**:
```bash
go build ./...
go test -v ./indexer/golang -run TestImplements
axon init --local .
axon query "SELECT s.name, i.name FROM (s:go:struct)-[:implements]->(i:go:interface) LIMIT 20"
# Should show e.g. sqlite.Storage implements graph.Storage
```

```go

    Debounce  time.Duration               // default: 150ms
    OnReindex func(path string, result *IndexResult, err error)
}
```

Add `Watch(ctx context.Context, path string, opts WatchOptions) error` to `Axon`:
1. Call `IndexWithOptions()` once to get initial state
2. Start `fsnotify.NewWatcher()`, add `path` recursively
3. Read events in a goroutine with debounce (collect events for `Debounce` duration, then
   find the common parent directory of all changed paths)
4. Call `IndexWithOptions()` with that parent directory as root
5. Call `opts.OnReindex` if set
6. Loop until `ctx.Done()`

## Phase F — Blast Radius CLI

### Task F1: Implement `axon impact` command

**Files created**: `cmd/axon/impact.go`  
**Files modified**: `cmd/axon/main.go`  
**Estimated time**: 35 minutes

**What to write**:  
```go
var impactCmd = &cobra.Command{
    Use:   "impact <symbol>",
    Short: "Show what would be affected if a symbol changes",
    RunE:  runImpact,
}
```

`runImpact` algorithm (pure AQL queries, no new storage):
1. Find the target node: `axon find --name <symbol>` equivalent — AQL query by name across
   go:func, go:struct, go:interface, go:method
2. Find all `go:ref` nodes referencing it (reverse `references` edges):
   `(ref:go:ref)-[:references]->(target)` — collect positions and packages
3. Group refs by containing package (join ref position's file path to `go:package` nodes)
4. Find packages that import those packages (reverse `imports` edges, requires Phase A)
5. Print formatted report: direct refs grouped by package, then indirect package dependents

**Output format**:
```
Impact analysis: Storage (go:interface)

Direct references (17):
  adapters/sqlite  12 refs  [call, type, field]
  cmd/axon          2 refs  [call]
  context           3 refs  [call]

Packages importing affected packages:
  axon        imports adapters/sqlite
  cmd/axon    imports adapters/sqlite, graph
```

**Verification**:
```bash
go build -o ./bin/axon ./cmd/axon
axon init --local .
./bin/axon impact Storage
# Should show impact report for graph.Storage interface
```
